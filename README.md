# Neha Mangla — Engineering Portfolio

A modern, responsive portfolio for Neha Mangla, a Senior Full Stack Engineer focused on React, TypeScript, product UI, dashboards, workflow-heavy interfaces, API-driven experiences, and tested frontend systems.

This portfolio is designed as a professional engineering brand site — clean, recruiter-readable, privacy-safe, and focused on product engineering quality.

## Live Demo

- Live Demo: https://neha-engineering-portfolio.vercel.app
- GitHub Repo: https://github.com/N-Mangla/neha-engineering-portfolio

## Highlights

- Premium SaaS/product-engineer visual design
- Responsive layout for desktop, tablet, and mobile
- Smooth motion using Framer Motion
- Reusable component structure
- Featured project case study
- Skills grouped by engineering area
- Public-safe experience summaries
- Certifications, education, achievements, and contact sections
- SEO metadata using Next.js App Router

## Tech Stack

- Next.js App Router

## Local Development

```bash
npm install
npm run dev
```

Open:

```txt
http://localhost:3000
```

## Production Build

```bash
npm run build
npm run start
```

## Deployment

This project is deployment-ready for Vercel using the default Next.js settings.

Recommended Vercel settings:

```txt
Framework Preset: Next.js
Build Command: npm run build
Output Directory: .next
Install Command: npm install
```

## Privacy Note

This portfolio intentionally avoids downloadable CV links, job-search language, relocation wording, visa-related wording, and confidential implementation details. It is designed as a professional engineering brand site that can be shared broadly.
